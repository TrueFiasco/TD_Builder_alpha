Parameters - Tune Page

Pre-Read Frames Prereadframes -  

Frame Timeout Frametimeout -  

File Open Timeout Opentimeout -  

The width and height of an image in pixels. Most TOPs, like the Movie File In TOP can set the image resolution. See Aspect Ratio for the width/height ratio of an image, taking into account non-square pixels.

The term "Frame" is a measurement of time used (1) in the Timeline, (2) as a time-unit in CHOPs, and (3) as a time unit in movie files that are read into TOPs and written out from TOPs. The frame rate is the frames per second (FPS).
