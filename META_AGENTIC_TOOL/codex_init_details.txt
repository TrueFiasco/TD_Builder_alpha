  5. Default Strategy

  Recommend: V2 (Improved Current) as default

  | Strategy | Quality | Tokens | Use Case                          |
  |----------|---------|--------|-----------------------------------|
  | V2 ✅    | 0.80    | ~28k   | Standard projects, balanced       |
  | V6       | 0.92    | ~52k   | Maximum quality, budget available |

  V2 gives 95% build success, 85% uniforms connected, 80% parameters functional. V6 is 99%/95%/90% but costs 2x tokens.

  Doc recommendation: Default to V2, mention V6 for "excellence" preset.

  6. Default Expertise Mode

  Recommend: compact=true as default

  | Mode            | Context Cost      | Use Case                     |
  |-----------------|-------------------|------------------------------|
  | compact=true ✅ | ~3.7K chars/cycle | Default for all KB queries   |
  | compact=false   | ~785K chars/cycle | Deep-dive on specific params |

  Compact mode gives 99.5% context savings, enables ~50 turns vs ~12 turns before context exhaustion.
