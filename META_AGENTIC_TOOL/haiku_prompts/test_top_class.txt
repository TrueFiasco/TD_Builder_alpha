## TOP_Class
Base class for all TOP (texture/image) operators in TouchDesigner.

### Members
- `width` → int (read-only): Texture width, measured in pixels.
- `height` → int (read-only): Texture height, measured in pixels.
- `aspect` → float (read-only): Texture aspect ratio, width divided by height.
- `aspectWidth` → float (read-only): Texture aspect ratio, width.
- `aspectHeight` → float (read-only): Texture aspect ratio, height.
- `depth` → int (read-only): Texture depth, when using a 3 dimensional texture.
- `gpuMemory` → int (read-only): The amount of GPU memory this TOP is using, in bytes.
- `curPass` → int (read-only): The current cooking pass iteration, beginning at 0.
- `isTOP` → bool (read-only): True if the operator is a TOP.
- `newestSliceWOffset` → int (read-only): When a Texture3D TOP fills its contents, tracks the newest slice filled.
- `pixelFormat` → str (read-only): Returns the pixel format of the TOP (e.g., RGBA8, RGBA32F).

### Methods
- `sample(x, y, z, u, v, w)` → Tuple[float, float, float, float]: Returns RGBA color value at the specified texture location. Use pixel coords (x,y,z) or normalized UV coords (u,v,w).
- `numpyArray(delayed=False, writable=False)` → numpy.ndarray: Returns the TOP image as a NumPy array for Python processing.
- `save(filepath, asynchronous=False, createFolders=False, quality=1.0)` → str: Saves the image to disk. Supports .tif, .jpg, .png, .exr, .dds formats.
- `saveByteArray(filetype, quality=1.0)` → bytearray: Saves the image to a bytearray in the specified format.
- `cudaMemory(stream=None)` → CUDAMemory: Copies TOP contents to CUDA memory for GPU processing.
