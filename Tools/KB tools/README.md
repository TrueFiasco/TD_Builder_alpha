# KB tools

The knowledge-base search tools on the **offline** `td-builder` server. All key-free.

`hybrid_search`, `get_operator_info`, `get_parameter_detail`, `query_graph`, `list_pop_operators`,
`find_operator_examples`, `find_operator_combination`, `find_parameter_usage`, `find_similar_networks`,
`get_network_patterns`.

Use these to look up operators, parameters, real examples, and common patterns before building.
Full signatures + inputs/outputs: [`../TOOLS.md`](../TOOLS.md).
