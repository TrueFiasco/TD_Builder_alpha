# Other tools

- `get_expert_prompt` — load a specialized TD expert's full prompt (td_designer, network_builder,
  td_glsl_expert, td_python_expert, ui_expert, critic) so the LLM can apply that expertise. Key-free.
- `get_server_info` — server identity: version, Python, resolved paths, and whether the live client
  is enabled. Useful for confirming which install is running.

Full signatures: [`../TOOLS.md`](../TOOLS.md).
