# Pre-Alpha 2-Week Schedule (suggested)

## Week 1 (Retrieval + grounding)
Day 1–2: Parameter lookups + encoding/runtime stability  
Day 3–4: Snippet retrieval quality (noise control, connection/dataflow emphasis)  
Day 5: Palette coverage (wiki summary joins, naming edge cases)  
Day 6–7: Consolidate fixes; re-run full prompt suite

## Week 2 (Builder push)
Day 8–9: Template-based JSON→toe/tox for a tiny “golden set” (2–3 templates)  
Day 10–11: Expand fixture coverage (more toxs / snippets) + regression checks  
Day 12: Failure-mode polish (keep expanded dir + toc + logs on failure)  
Day 13–14: Freeze + re-run suite; decide default build output (python vs toe/tox)
